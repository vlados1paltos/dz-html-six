Теоретическая часть:
1. Назовите основной тег, применяемый для создания таблиц, и объясните его назначение.
Ответ: 
Основной тег, применяемый для создания таблиц - это тег table.
Он используется для создания html-таблицы и служит контейнером для всех её элементов.

2. Перечислите основные семантические элементы таблицы в порядке их логического расположения и объясните их назначение:
Ответ:
1) thead - он содержит заголовочную часть таблицы.
2) tbody - это основное тело таблицы, где находятся данные.
3) tfoot - это нижняя часть таблицы, чаще всего она используется для подведения итогов или любой другой дополнительной информации.
Внутри этих секций используются:
1) tr - это строка таблицы. Объединяет ячейки в одну горизонтальную линию.
2) th - это ячейка-заголовок. Часто используется в thead. Эта ячейка выделяется жирным и центрируется по умолчанию.
3) td - это обычная ячейка таблицы, содеражащая в себе те или иные данные.

3. Объясните, будет ли браузер интерпретировать как ошибку размещение тега tfoot в самом верху, и почему это допустимо в html5.
Ответ:
Нет, браузер не будет интерпретировать это как ошибку.
В html5 допускается размещение тега tfoot перед tbody и даже сразу после thead.
Это связано с тем, что браузер при разборе html автоматически корректирует структуру таблицы и логически размещает tfoot внизу при отображении.
